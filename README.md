Python Scientific Calculator
A sleek, resizable scientific calculator built with Python's standard Tkinter library. It features a modern dark theme inspired by Google's Material Design (Motorola-style) and supports complex mathematical expressions, including trigonometric functions and parentheses.

++ Features
Modern UI: Clean, dark-themed interface that is easy on the eyes.

Fully Resizable: The window and all buttons scale fluidly to any size.

Scientific Functions: Includes:

Sine (sin)

Cosine (cos)

Tangent (tan)

Square Root (√)

Complex Expressions: Full support for parentheses () to manage the order of operations.

Standard Operations: +, -, ×, ÷

Essential Controls:

AC: All Clear to reset the expression.

⌫: Backspace to delete the last character.

Safe Evaluation: Uses a secured eval() environment to safely parse and compute complex mathematical strings, preventing any security risks.

++ Built With
Python 3

Tkinter: Python's standard, built-in GUI library.

Math: Python's standard library for mathematical functions.

++ Getting Started
No external libraries or packages are required! You only need a standard installation of Python 3.
